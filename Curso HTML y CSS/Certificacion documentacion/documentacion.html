Introduccion a Python

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Fue creado por Guido van Rossum y su primera versión fue lanzada en 1991. Python se ha convertido en uno de los lenguajes de programación más populares del mundo debido a su simplicidad, legibilidad y versatilidad.

Se han creado gran variedad de aplicaciones que han sido desarrollados con el interprete de python, entre ellas estan: Pinterest, Dropbox, Spotify, Netflix, entre otras.

  • Simplicidad y legibilidad: Python está diseñado para ser fácil de leer y escribir, con una sintaxis clara y concisa que hace que el código sea más comprensible y mantenible.
  • Multiparadigma: Python admite varios paradigmas de programación, incluidos el orientado a objetos, el procedural y el funcional.
  • Bibliotecas extensas: Python cuenta con una amplia biblioteca estándar que proporciona módulos y paquetes para realizar tareas comunes, como manipulación de cadenas, manejo de archivos, comunicación de red y pruebas.
  • Portabilidad: Python es multiplataforma, lo que significa que el código escrito en Python puede ejecutarse en diferentes sistemas operativos sin cambios.
  • Comunidad activa: Python tiene una gran comunidad de desarrolladores que contribuyen con bibliotecas, herramientas y soporte, lo que facilita el aprendizaje y la resolución de problemas.
Sentencia if

Sintaxis if:

La sentencia if en Python se utiliza para tomar decisiones en el código basado en ciertas condiciones. Es una de las estructuras de control más fundamentales en cualquier lenguaje de programación y permite ejecutar diferentes bloques de código dependiendo del resultado de una condición booleana (verdadera o falsa).

if condición:
# Bloque de código a ejecutar si la condición es verdadera

Uso de else:

La sentencia else se utiliza para ejecutar un bloque de código cuando la condición del if es falsa.

edad = 16

if edad >= 18:
ㅤㅤprint("Eres un adulto.")
else:
ㅤㅤprint("Eres menor de edad.")

En este ejemplo, el mensaje "Eres un adulto." se imprimirá porque la condición edad >= 18 es verdadera.

Uso de elif:

La sentencia elif (abreviatura de "else if") se utiliza para verificar múltiples condiciones. Si la primera condición es falsa, Python verifica la siguiente condición elif, y así sucesivamente.

edad = 16

if edad >= 18:
ㅤㅤprint("Eres un adulto.")
elif edad >= 13:
ㅤㅤprint("Eres un adolescente.")
else:
ㅤㅤprint("Eres un niño.")

En este ejemplo, el mensaje "Eres un adolescente." se imprimirá porque la condición edad >= 18 es falsa, pero la condición edad >= 13 es verdadera.

Anidacion del if

Es posible anidar sentencias if, lo que significa que puedes colocar una sentencia if dentro de otra. Esto permite manejar condiciones más complejas.

edad = 20
tiene_identificacion = True

if edad >= 18:
ㅤㅤif tiene_identificacion:
ㅤㅤㅤㅤprint("Puedes entrar al club.")
ㅤㅤelse:
ㅤㅤㅤㅤprint("Necesitas una identificación para entrar.")
else: print("No puedes entrar al club porque eres menor de edad.")

En este caso, el mensaje "Puedes entrar al club." se imprimirá porque ambas condiciones edad >= 18 y tiene_identificacion son verdaderas.

Sentencia for

La sentencia for en Python se utiliza para iterar sobre una secuencia (como una lista, una tupla, un diccionario, un conjunto o una cadena) o cualquier otro objeto iterable. Esta sentencia permite ejecutar un bloque de código varias veces, una vez por cada elemento en la secuencia.

Estructura basica del bucle for:

 for variable in secuencia:
ㅤㅤ# Código a ejecutar en cada iteración

Ejemplos con el bucle for:

Iterar sobre una lista:

frutas = ["manzana", "banana", "cereza"]

for fruta in frutas:
ㅤㅤprint(fruta)

Iterar sobre una cadena:

mensaje = "Hola"

for letra in mensaje:
ㅤㅤprint(letra)
Sentencia range

La función range() en Python se utiliza para generar una secuencia de números. Es especialmente útil cuando se combina con bucles for para iterar un número específico de veces o para generar secuencias de números con pasos definidos.

La funcion range se puede utilizar de diferentes maneras:

Sintaxis basica del range:

  • Un solo argumento: range(stop)
  • Dos argumentos: range(start, stop)
  • Tres argumentos: range(start, stop, step)

Con un solo argumento:

Cuando se usa range(stop), se genera una secuencia de números desde 0 hasta stop-1.

for i in range(5):
ㅤㅤprint(i)

#Salida
0
1
2
3
4

Con dos argumentos:

Cuando se usa range(start, stop), se genera una secuencia de números desde start hasta stop-1.

for i in range(2, 6):
ㅤㅤprint(i)

#Salida
2
3
4
5

Con tres argumentos:

Cuando se usa range(start, stop, step), se genera una secuencia desde start hasta stop-1, incrementando (o decrementando) en step.

for i in range(1, 10, 2):
ㅤㅤprint(i)

#Salida
1
3
5
7
9
Sentencia pass

La sentencia pass en Python es una operación nula; es decir, cuando se ejecuta, no sucede nada. Se utiliza como un marcador de posición en el código donde se requiere una declaración sintácticamente, pero no se necesita ninguna operación o acción. Esto es útil en varias situaciones, como la definición de funciones, clases, bucles o condiciones que aún no se han implementado.

Ejemplos de la sentencia pass:

Dentro de funciones:

Cuando se define una función que aún no tiene una implementación, se puede usar pass para permitir que el código se ejecute sin errores de sintaxis.

def funcion_no_implementada():
ㅤㅤpass

# Llamar a la función no hace nada
funcion_no_implementada()

En bucles:

Si se necesita un bucle que no haga nada (tal vez como un marcador de posición mientras se desarrolla el código), pass se puede usar dentro del bucle.

for i in range(5):
ㅤㅤpass
Sentencia break continue y else

En Python, break, continue y pass son sentencias de control de flujo que se utilizan para manejar la ejecución de bucles y otros bloques de código de manera específica. A continuación, te proporcionaré una introducción a cada una de estas sentencias con ejemplos.

Sentencia break:

La sentencia break se utiliza para salir de un bucle de forma anticipada, es decir, antes de que se haya completado normalmente. Esto es útil cuando se quiere terminar un bucle al cumplirse una condición específica.

for i in range(10):
ㅤㅤif i == 5:
ㅤㅤㅤㅤbreak
ㅤㅤprint(i)

#Salida
0
1
2
3
4

En este ejemplo, el bucle for se detiene cuando i es igual a 5.

Sentencia continue:

La sentencia continue se utiliza para omitir el resto del código dentro de un bucle en la iteración actual y pasar a la siguiente iteración. Esto es útil cuando se desea saltar ciertas iteraciones basadas en una condición.

for i in range(10):
ㅤㅤif i % 2 == 0:
ㅤㅤㅤㅤcontinue
ㅤㅤprint(i)

#Salida
1
3
5
7
9

Sentencia else

La sentencia else en Python es una estructura de control de flujo que se utiliza comúnmente junto con las sentencias if, for, y while. Su función principal es definir un bloque de código que se ejecuta cuando la condición en la sentencia if es falsa, o cuando un bucle for o while se completa normalmente (es decir, sin que se encuentre una sentencia break).

Uso de else con if:

x = 10

if x > 15:
ㅤㅤprint("x es mayor que 15")
elif x > 5:
ㅤㅤprint("x es mayor que 5 pero menor o igual a 15")
else:
ㅤㅤprint("x es menor o igual a 5")

#Salida
x es mayor que 5 pero menor o igual a 15

En este ejemplo, la primera condición x > 15 es falsa, pero la segunda condición x > 5 es verdadera, por lo que se ejecuta el bloque correspondiente al elif.

Uso de else con for:

La sentencia else también se puede utilizar con bucles for. En este contexto, el bloque else se ejecuta después de que el bucle ha iterado sobre todos los elementos de la secuencia. Sin embargo, si el bucle se interrumpe con una sentencia break, el bloque else no se ejecuta.

for i in range(5):
ㅤㅤif i == 3:
ㅤㅤㅤㅤbreak
ㅤㅤprint(i)
else:
ㅤㅤprint("El bucle ha terminado normalmente")

print("Fin del primer bucle")

for i in range(5):
ㅤㅤprint(i)
else:
ㅤㅤprint("El bucle ha terminado normalmente")

#Salida
0
1
2
Fin del primer bucle
0
1
2
3
4
El bucle ha terminado normalmente

En este ejemplo:

  • En el primer bucle, se encuentra una sentencia break cuando i es igual a 3, por lo que el bucle se interrumpe y el bloque else no se ejecuta.
  • En el segundo bucle, no hay sentencia break, por lo que después de iterar todos los elementos, se ejecuta el bloque else
